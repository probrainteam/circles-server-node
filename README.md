> # Yarn
>
> - yarn이 없다면 yarn 설치 후 진행해주세요
> - mac :: brew install yarn
>
> # local 세팅
>
>       git clone ${주소} && yarn install
>
> # 서버 구동 (with nodemon)
>
> ## yarn main
>
> - 배포 서버를 위한 구동
> - AWS db, domain setting으로 서버가 구동됩니다 (진행중)
>
> ## yarn dev
>
> - 로컬 서버를 위한 구동
> - localhost db, domain setting으로 서버가 구동됩니다.
> - 이 과정에서 mysql 오류가 날 수 있습니다.
